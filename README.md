# Tip Calculator 💰

This is my second Python project from Dr. Angela Yu's 100 Days of Code Bootcamp.

## Description

This program calculates how much each person should pay after adding a tip.

The user enters:
- Total bill amount
- Tip percentage (10%, 12%, or 15%)
- Number of people sharing the bill

The program calculates and displays the amount each person should pay.

## Concepts Learned

- Variables
- User Input
- Mathematical Calculations
- Float Data Type
- Rounding Numbers
- f-Strings
